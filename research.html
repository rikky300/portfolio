1. 機械学習の基礎

機械学習の目標は、データから関数 $f$ を学習することである。入力 $x$ に対して出力 $y$ を返す対応 $y = f(x)$ のパラメータを、データをもとに最適化する。

例えば線形回帰では $y = ax + b$ のパラメータ $a, b$ を求めたい。$N$ 個のデータ $\{(x_i, y_i)\}$ に対して損失関数として平均二乗誤差（MSE）を定義する。

$$L(a, b) = \frac{1}{N} \sum_{i=1}^{N} \bigl(y_i - (ax_i + b)\bigr)^2$$

この損失を最小化するために勾配降下法を用いる。パラメータ $a$ に関する勾配は

$$\frac{\partial L}{\partial a} = -\frac{2}{N} \sum_{i=1}^{N} x_i \bigl(y_i - (ax_i + b)\bigr)$$

となり、学習率 $\lambda$ を用いてパラメータを更新する。

$$a \leftarrow a - \lambda \frac{\partial L}{\partial a}$$

これを繰り返すことで損失が小さくなる方向にパラメータが更新される。

2. ニューラルネットワークとディープラーニング

線形モデルでは表現できない非線形な関係を学習するために、パーセプトロンを多層に積み重ねた多層パーセプトロン（MLP）が登場した。$L$ 層のネットワークを以下のように表す。

$$\boldsymbol{h}^{(l)} = \sigma\!\left(\boldsymbol{W}^{(l)} \boldsymbol{h}^{(l-1)} + \boldsymbol{b}^{(l)}\right)$$

ここで $\boldsymbol{W}^{(l)}$ は重み行列、$\boldsymbol{b}^{(l)}$ はバイアス、$\sigma$ は ReLU などの活性化関数である。パラメータの更新には誤差逆伝播法（バックプロパゲーション）を用いて連鎖律で勾配を効率よく計算する。

CNN（畳み込みニューラルネットワーク）

画像に対しては全結合層ではなく畳み込み層を用いる。フィルタを空間的に共有することで局所的な特徴（エッジ・テクスチャなど）を効率よく捉え、画像分類の精度が飛躍的に向上した。

3. 生成モデル（VAE・拡散モデル）

生成モデルの目標は、自然画像の真の確率密度関数 $p(\boldsymbol{x})$ を近似し、そこからサンプリングすることで新しい画像を生成することである。

VAE（変分オートエンコーダ）[1]

潜在変数 $\boldsymbol{z}$ を経由して画像を生成するモデル。周辺尤度 $p(\boldsymbol{x})$ の最大化は計算困難なため、代わりに Evidence Lower Bound（ELBO）を最大化する。

$$\mathcal{L}_{\text{ELBO}} = \mathbb{E}_{q_\phi(\boldsymbol{z}|\boldsymbol{x})}\!\left[\log p_\theta(\boldsymbol{x}|\boldsymbol{z})\right] - D_{\mathrm{KL}}\!\left(q_\phi(\boldsymbol{z}|\boldsymbol{x}) \,\|\, p(\boldsymbol{z})\right)$$

第1項は再構成誤差（デコーダの性能）、第2項はエンコーダの出力分布を事前分布 $p(\boldsymbol{z}) = \mathcal{N}(\boldsymbol{0}, \boldsymbol{I})$ に近づける正則化項である。生成時は $\boldsymbol{z} \sim \mathcal{N}(\boldsymbol{0}, \boldsymbol{I})$ からサンプリングし、学習済みデコーダに通す。

拡散モデル（Diffusion Model）

拡散モデルには「スコアベース」と「熱力学着想」の2つの系譜があり、現在は融合して発展している。

スコアベース [2]：確率密度そのものではなくスコア $\nabla_{\boldsymbol{x}} \log p(\boldsymbol{x})$ をニューラルネットで近似し、Langevin ダイナミクスでサンプリングする。

$$\boldsymbol{x}_{t+1} = \boldsymbol{x}_t + \epsilon\, \nabla_{\boldsymbol{x}} \log p(\boldsymbol{x}_t) + \sqrt{2\epsilon}\, \boldsymbol{z}_t, \quad \boldsymbol{z}_t \sim \mathcal{N}(\boldsymbol{0}, \boldsymbol{I})$$

単一ノイズレベルでは確率密度が薄い領域でスコアの推定が不安定になるため、複数ノイズレベル $\{\sigma_i\}$ でスコアを学習し、ノイズレベルを大から小へ徐々に下げながらサンプリングする。

DDPM（熱力学着想）[3]：クリーンな画像 $\boldsymbol{x}_0$ に段階的にガウスノイズを加える拡散過程を定義する。

$$q(\boldsymbol{x}_t \mid \boldsymbol{x}_0) = \mathcal{N}\!\left(\boldsymbol{x}_t;\, \sqrt{\bar{\alpha}_t}\,\boldsymbol{x}_0,\, (1 - \bar{\alpha}_t)\boldsymbol{I}\right)$$

ここで $\bar{\alpha}_t = \prod_{s=1}^{t}(1-\beta_s)$ はノイズスケジュールによって決まる係数。学習目標は加えられたノイズ $\boldsymbol{\epsilon}$ の推定であり、目的関数は

$$\mathcal{L} = \mathbb{E}_{\boldsymbol{x}_0,\, \boldsymbol{\epsilon},\, t}\left[\left\|\boldsymbol{\epsilon} - \boldsymbol{\epsilon}_\theta\!\left(\boldsymbol{x}_t,\, t\right)\right\|^2\right]$$

となる。推論時は純粋なノイズ $\boldsymbol{x}_T \sim \mathcal{N}(\boldsymbol{0}, \boldsymbol{I})$ から逆過程をたどることで画像を生成する。

4. 画像復元と最適化

手ブレ・ノイズなどで劣化した観測画像 $\boldsymbol{y}$ から元画像 $\boldsymbol{x}^*$ を推定する問題を考える。劣化モデルは

$$\boldsymbol{y} = \boldsymbol{A}\boldsymbol{x}^* + \boldsymbol{n}$$

と表される。ここで $\boldsymbol{A}$ は劣化行列（ぼかし・ダウンサンプリングなど）、$\boldsymbol{n} \sim \mathcal{N}(\boldsymbol{0}, \sigma^2 \boldsymbol{I})$ はガウスノイズである。

元画像の推定は次の最適化問題として定式化される。

$$\hat{\boldsymbol{x}} \in \operatorname*{argmin}_{\boldsymbol{x}} \left( \underbrace{\frac{1}{2}\|\boldsymbol{y} - \boldsymbol{A}\boldsymbol{x}\|^2}_{\text{データ忠実項}} + \underbrace{g(\boldsymbol{x})}_{\text{正則化項}} \right)$$
  • データ忠実項：観測 $\boldsymbol{y}$ との整合性を保つ
  • 正則化項 $g(\boldsymbol{x})$：画像の先験情報（自然画像らしさ）を解に組み込む

$g(\boldsymbol{x})$ の設計方針によって大きく2つのアプローチに分かれる。

  • Model-driven：$\ell_1$ ノルム（スパース性）など数式で定義された正則化項を使う。解釈性・収束保証に優れる。
  • Data-driven：劣化画像 → 復元画像を End-to-End の DNN で学習する。精度は高いが最適化の観点での解釈性が低い。

5. Plug-and-Play と提案手法

Plug-and-Play（PnP）[4]

Model-driven の枠組みを保ちつつ、正則化項の近接写像（proximal operator）を DNN デノイザーで置き換える手法が Plug-and-Play である。ADMM や HQS などのアルゴリズムに組み込まれ、解釈性と精度を両立する。

HQS（Half Quadratic Splitting）では補助変数 $\boldsymbol{z}$ を導入し、

$$\boldsymbol{z}^{(k+1)} = \operatorname{Denoiser}_\sigma\!\left(\boldsymbol{x}^{(k)}\right), \qquad \boldsymbol{x}^{(k+1)} = \left(\boldsymbol{A}^\top \boldsymbol{A} + \mu \boldsymbol{I}\right)^{-1}\!\left(\boldsymbol{A}^\top \boldsymbol{y} + \mu\, \boldsymbol{z}^{(k+1)}\right)$$

と交互に更新する。デノイザーを DNN に置き換えることで高精度な復元が可能になる。

提案手法：解釈性を持つ拡散モデルの PnP 組み込み

拡散モデルを PnP のデノイザーとして組み込む手法が近年提案されているが、確率的サンプリングがメインであり最適化問題としての解釈が不明瞭である。医療画像など信頼性が求められる場面では、「この復元結果は数学的にどのような問題の解か」が重要になる。

提案手法では、Hurault et al. [5] が提案した Gradient Step Denoiser の枠組みを拡散モデルに拡張する。具体的には拡散モデルのデノイザーに以下の2つの制約を課す。

  • 勾配制約：デノイザー $D_\theta(\boldsymbol{x})$ がポテンシャル関数 $\phi(\boldsymbol{x})$ の勾配として表せること。すなわち $D_\theta(\boldsymbol{x}) = \nabla_{\boldsymbol{x}} \phi(\boldsymbol{x})$
  • Lipschitz 制約：ネットワークの Lipschitz 定数 $L$ が有界であること。
    $$\|D_\theta(\boldsymbol{x}) - D_\theta(\boldsymbol{x}')\| \leq L \|\boldsymbol{x} - \boldsymbol{x}'\| \quad \forall\, \boldsymbol{x}, \boldsymbol{x}'$$

この2つを満たすことで、最適化の観点から収束保証のある PnP フレームワークに拡散モデルを組み込むことができる。Lipschitz 制約はネットワークの各層に NablaResNet 構造と重みのスペクトル正規化を組み合わせて実装している。

結果と課題

制約なしの既存手法 DPIR [6] と比較して同等以上の復元性能を達成した。一方で、罰則関数によるソフト制約のため学習が進むにつれて Lipschitz 定数が境界を超えるケースが発生する。現在はハードな制約を維持しつつ表現力も上げる学習手法を試行中。

現在ジャーナル投稿に向けて、シミュレーション実験と Introduction のブラッシュアップに取り組んでいる。

6. LLM / VLM（勉強中）

食事解析アプリ Zubora での Gemini API 活用をきっかけに、大規模言語モデル（LLM）・Vision-Language Model（VLM）の仕組みと応用に興味を持って学習中。Transformer のアーキテクチャや Attention 機構の基礎から、RAG・プロンプトエンジニアリングの実践まで幅広く調べている。

今夏のインターンなどを通じて実際に扱う機会を増やし、アプリの解析精度改善にも直結させていく予定。
