研究テーマ

画像復元（Image Restoration）に拡散モデルを応用する研究に取り組んでいる。手ブレやノイズなどで劣化した画像から元の画像を推定する問題に対し、数学的な解釈性を保ちながら拡散モデルを組み込む手法の開発を目指している。

理論寄りの研究室に所属しており、実装に多くの時間を割いている。純粋な理論よりも、その理論をどう実用化するかを考えることに面白さを感じており、Claude Code などの AI コーディングツールを活用してアイデアをすぐ実装・検証するサイクルが自分のスタイルに合っている。

機械学習の基礎

線形・非線形モデルと勾配降下法

データから関数を学習すること——入力 x に対して出力 y を返す f(x) のパラメータを最適化する——が機械学習の基本的な枠組みである。回帰・分類などのタスクに対して損失関数（平均二乗誤差など）を定義し、勾配降下法でパラメータを更新する。

ニューラルネットワークとディープラーニング

パーセプトロンによって特徴量の重みづけを自動化し、多層化（多層パーセプトロン）によって非線形関数も学習できるようになった。誤差逆伝播法で効率よく勾配を計算し、活性化関数の工夫・GPU の活用・データ量の増加などがディープラーニングの発展を支えた。

CNN（畳み込みニューラルネットワーク）の登場により、画像の局所的な特徴を効率よく捉えられるようになり、画像分類の精度が大幅に向上した。

生成モデルと拡散モデル

VAE（変分オートエンコーダ）

教師なし学習で、入力を潜在空間に圧縮してから再構成する過程を学習することで画像を生成する。エンコーダが入力を正規分布に近い状態に圧縮し、デコーダがその分布からサンプリングして元の画像を生成するよう学習される。通常のオートエンコーダと異なり、新しい画像の生成が可能。

拡散モデル（Diffusion Model）

2つの系譜——スコアベースと熱力学着想——が融合して発展した生成モデル。

スコアベース：確率密度関数そのものではなくその勾配（スコア）を学習し、スコアに沿ってサンプリングすることで画像を生成する。複数ノイズレベルでスコアを学習することで、確率密度が薄い領域でも正しく進める。

熱力学系譜（DDPM など）：クリーンな画像に段階的にノイズを加えてゆく拡散過程を定義し、その逆過程（デノイジング）を学習する。学習済みモデルに純粋なノイズを入力することで新しい画像を生成できる。

画像復元と提案手法

画像復元の問題設定

カメラの手ブレなどで劣化した観測画像 y から、元画像 x* を推定する問題。y = Ax* + n（A: 劣化行列、n: ノイズ）と定式化し、データ忠実項（観測との整合性）と正則化項（画像の先験情報）の和を最小化する最適化問題として解く。

Plug-and-Play と提案アプローチ

数式で定義された正則化項を用いる Model-driven 手法は解釈性・収束保証に優れ、DNN を使う Data-driven 手法は精度が高い。Plug-and-Play はこれらを組み合わせ、DNN をデノイザーとして最適化の枠組みに組み込む手法である。

提案手法では、この Plug-and-Play に拡散モデルを組み込む。ただし既存手法は確率的サンプリングが主体で数学的解釈性に乏しい。そこで「ネットワークが勾配の形になること」と「Lipschitz 定数を満たすこと」の2つの制約を課した拡散モデルを学習し、数学的に良い性質を保ちながら高品質な画像復元を実現した。

制約のない既存手法（DPIR など）と同等以上の復元性能を達成。現在は制約を完全に満たしつつ表現力も維持するための学習手法を研究中。

興味として学んでいること

LLM / VLM

食事解析アプリ Zubora での Gemini API 活用をきっかけに、LLM・VLM の仕組みと応用に興味を持ち、学習を進めている。今夏のインターンなどを通じて実際に扱う機会を増やし、理解を深めたいと考えている。理解が進めばアプリの解析精度改善にも直結するため、調べながら着実に実装に反映していく予定。
